# TRPO_lab_widgets

Коллекция виджетов для лабораторных работ 

## Структура проекта

- **`counter/`** — Виджет-счётчик с возможностью увеличения/уменьшения значения и сброса. Иллюстрирует базовую работу с сигналами/слотами и QML-компонентами.

- **`rotation/`** — Виджет для управления поворотом геометрических фигур.

- **`lab_widgets/`** — виджет для возведения чисел в квадрат.

## Сборка и запуск

Проект написан на **C++** с использованием **Qt 5.15.2**
